The Holy Grail Layout

The Holy Grail layout is a common web page structure with three columns: a main content area (center), and sidebars (left and right).

Box Model in CSS

The box model defines how elements are rendered on a web page. It consists of:

  • Content: The actual content (text, images) inside the element.
  • Padding: Space between the content and the border.
  • Border: A line around the padding and content.
  • Margin: Space outside the border.

Clearfix Hack

When floating elements, their parent container may not expand to accommodate them. The clearfix hack ensures proper container height by adding an empty pseudo-element with clear: both;.

Margin Collapse

Vertical margins of adjacent elements can collapse into a single margin. Use margin: auto; for centering and negative margins for special effects.

Problem with Holy Grail Layout

The challenge was achieving equal-height columns with separate backgrounds and fixed-width sidebars. CSS limitations and browser inconsistencies made it tricky.

Modern Solutions

Today, we have robust solutions like Flexbox and CSS Grid Layout. They allow flexible, responsive Holy Grail layouts without workarounds.
